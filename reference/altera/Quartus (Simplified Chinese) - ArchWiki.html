What Quartus

Quartus is [ Altera ] has introduced, for Altera's programmable device design kit. Full range of products available for the Altera synthesis, adaptation, simulation, download. The current latest version is 9.0SP1, for Windows, Linux, SunOS and other platforms. For the Windows platform, the online edition free for all platforms for one month free subscription version. Specifically refer to the official website!

Download

Altera Quartus II official website can be downloaded from the download address: [ [1] ]

  • Prepare a free space is larger than 7G in the space for download and extract the Quartus: 
$ Mkdir ~ / quartus
  • Download Quartus II 9.0 For Linux 
$ Wget ftp://ftp.altera.com/outgoing/release/90_quartus_linux.tar

This will download a very long long time, think about the ordinary on-line ADSL 300K Vs 3000MB, so this time you can go out to eat, and eat almost back to the download finished.

  • Dependency, Quartus using tcsh, so it is necessary to install tcsh 
# Pacman-S tcsh
  • Installation 
$ Cd ~ / quartus #. / Install
  • Note the need to use root privileges to install, the installation process is very simple, you need to answer three questions:
  1. Installation location, the default is / opt/altera9.0 /
  2. Device simulation library selection, the default is all the libraries, you can choose one or a few, such as cycii cyciii on behalf of select cyclone II and cyclone III
  3. Read the license file

Here, on the installation is completed, but this time can not be used immediately. Treatment must be the next step, if not addressed, there will be [ weird bugs ]

Treatment after the installation

  • First, remove the / etc / issue inputrc set for special characters (for the agent to start automatically clear screen): 
echo "Arch Linux \ r (\ n) (\ l)"> / etc / issue
  • Then, check whether your host name added to / etc / hosts, if not we must add 
echo "127.0.0.1 myhost.localdomain myhost">> / etc / hosts

Your own IP address and host name and replace the above myhost 127.0.0.1

Abort (aborted)

This is because you can not resolve the host name to IP address caused. In this case you can continue to build on the command line, no problem. GUI startup will fail, however. In accordance with the above mentioned will be your host name and IP address records added to the / etc / hosts can be.
